Introduction to Rust Programming

What is Rust?

Rust is a modern systems programming language that focuses on safety, speed, and concurrency. Developed by Mozilla, it eliminates many of the common pitfalls of other low-level languages like C and C++ by providing memory safety guarantees without needing a garbage collector.

  • Memory Safety: Rust ensures memory safety through its ownership and borrowing system, preventing data races and null pointer dereferences.
  • Performance: Rust achieves performance on par with C and C++, with fine control over memory usage and system resources.
  • Concurrency: Rust simplifies writing safe concurrent code, avoiding common concurrency issues.

Why Learn Rust?

Rust is increasingly becoming the go-to language for performance-critical applications such as game engines, web assembly, embedded systems, and large-scale systems programming.

Key benefits of Rust include:

  • Strong focus on safety and security.
  • High performance with minimal runtime overhead.
  • Great for concurrency and parallelism.
  • Growing community with excellent tooling and documentation.

Basic Syntax and Example Programs

Hello, World!

The simplest program in Rust is the "Hello, World!" program, demonstrating basic syntax and how to print text to the console.

// hello.rs
fn main() {
    println!("Hello, world!");
}

In this program:

  • fn main() is the main function where the execution starts.
  • println! is a macro to print text to the console.

Variables and Mutability

Rust variables are immutable by default. To make a variable mutable, use the mut keyword.

// variables.rs
fn main() {
    let x = 5; // Immutable variable
    println!("x = {}", x);

    let mut y = 10; // Mutable variable
    println!("y = {}", y);
    
    y = 15;
    println!("y is now = {}", y);
}

Data Types and Functions

Rust has strong static typing. Here's an example of declaring variables with different data types and calling functions.

// data_types.rs
fn main() {
    let a: i32 = 10; // Integer type
    let b: f64 = 20.5; // Floating point type
    let is_active: bool = true; // Boolean type
    let name: &str = "Rust"; // String type

    println!("Integer: {}, Float: {}, Boolean: {}, Name: {}", a, b, is_active, name);

    let result = add(5, 10);
    println!("5 + 10 = {}", result);
}

// A simple function that adds two integers
fn add(x: i32, y: i32) -> i32 {
    x + y
}

Control Flow

Rust provides control flow constructs such as if and loops like for, while, and loop.

// control_flow.rs
fn main() {
    let number = 6;

    if number % 2 == 0 {
        println!("The number is even");
    } else {
        println!("The number is odd");
    }

    for i in 0..5 {
        println!("i = {}", i);
    }

    let mut counter = 0;
    while counter < 5 {
        println!("Counter = {}", counter);
        counter += 1;
    }
}

Ownership and Borrowing

Ownership is one of the key concepts in Rust, ensuring memory safety without a garbage collector. Here's a simple example of ownership and borrowing:

// ownership.rs
fn main() {
    let s1 = String::from("Hello");
    let s2 = &s1; // Borrowing s1

    println!("s1 = {}, s2 = {}", s1, s2);
